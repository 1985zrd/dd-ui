# dd-ui-vue 

### 学习ui组件库
> 主要用来学习交流

- 环境依赖   node v10.0.0+ 

---

### 项目地址

[dd-ui](https://github.com/1985zrd/dd-ui.git)

---

## 安装
`npm i -S dd-ui-vue`

## 使用
```js
import ddUi from 'dd-ui-vue'
import 'dd-ui-vue/lib/style.css'

Vue.use(ddUi)
```
